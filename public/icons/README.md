# App Icons

Place your PWA icons here. Required sizes:

- `icon-16x16.png` - Favicon (small)
- `icon-32x32.png` - Favicon (medium)
- `icon-72x72.png` - PWA icon
- `icon-96x96.png` - PWA icon
- `icon-128x128.png` - PWA icon
- `icon-144x144.png` - PWA icon
- `icon-152x152.png` - PWA icon
- `icon-192x192.png` - PWA icon (required)
- `icon-384x384.png` - PWA icon
- `icon-512x512.png` - PWA icon (required)
- `apple-touch-icon.png` - iOS icon (180x180)

## Generate Icons

You can generate these icons from a single source image using tools like:

1. **RealFaviconGenerator** - https://realfavicongenerator.net/
2. **PWA Asset Generator** - `npx pwa-asset-generator logo.png ./public/icons`
3. **Favicon.io** - https://favicon.io/

## Source Image Recommendations

- Use a square PNG image (512x512 or larger)
- Use simple, bold design for smaller sizes
- Ensure the icon is visible on both light and dark backgrounds
